# Medical Lab Management System

## 📋 Project Overview
A comprehensive web application for managing patient bookings, lab tests, sample collections, and medical reports.

## 🛠️ Tech Stack
- **Backend:** PHP 8.0+
- **Database:** MySQL 8.0+
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Architecture:** MVC Pattern

## ✨ Features
- Patient Registration & Management
- Doctor Management
- Lab Technician Operations
- Online Appointment Booking
- Payment Processing
- Sample Collection Tracking
- Lab Test Management
- Medical Report Generation
- Admin Dashboard

## 📊 Database Entities
Based on ER Diagram:
- Person (Base entity)
- Patient (extends Person)
- Doctor
- Lab_Technician
- Booking
- Payment
- Sample_Collection
- Test
- Test_Report
- Report_Details
